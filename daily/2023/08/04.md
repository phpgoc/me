> c/cpp 古时可以用的写法，现在依然可以用
```cpp
#include <iostream>

int main() <%
  int c<:2:> = <%1, 2%>;
  std::cout << c<:0:> << std::endl;
  std::cout << c<:1:> << std::endl;
  return 0;
%>
```
