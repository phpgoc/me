> svelte promise
```html

{#await promise}
<p>
	 ...waiting
</p>
{:then number}
<p>
	The number is {number}
</p>
{:catch error}
<p style="color:red">
	{error.message}
</p>
{/await}
```

> svelte 组件事件向外传递
```html
```javascript
import { createEventDispatcher } from 'svelte';

	const dispatch = createEventDispatcher();

	function sayHello() {
		dispatch("message",{
			text: 'Hello!'
		});
	}
```
